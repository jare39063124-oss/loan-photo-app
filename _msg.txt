fix(v3.4.0): three critical fixes

1. Status bar overlay fix:
   - Add status bar height spacer at top of all screens
   - Call setDecorFitsSystemWindows(true) on Android API 30+
   - Set status bar color to dark theme color with light icons
   - Intercept Android BACK button (key=27): settings->main, main->welcome,
     welcome double-tap to exit. Edge swipe no longer quits app directly.

2. Touch target size increase:
   - All buttons: min height 52-60dp (was 40-48dp)
   - Spinner/dropdown rows: 52dp height with 15sp font
   - Title bars: 56dp height, settings back button 52dp
   - RowWidget (list items): 80dp height, 16sp name font
   - Section labels: 17sp, 40dp height
   - Photo type popup buttons: 72dp height, 16sp font

3. Excel file PermissionError fix (scoped storage):
   - Replace plyer.filechooser with native Android SAF file picker
   - Use Intent.ACTION_OPEN_DOCUMENT + ContentResolver
   - Copy selected file to app-private dir via InputStream before reading
   - This eliminates errno 13 PermissionError on Android 10+
   - Grant persistable URI permission for reliable access
   - Removed unnecessary READ_MEDIA_IMAGES/READ_EXTERNAL_STORAGE
     (SAF doesn't need them - user explicitly grants access per file)

4. Cleanup:
   - Removed duplicate font files (assets/simhei.ttf was redundant copy)
   - Removed broken placeholder fonts (28KB NotoSansSC.ttf etc.)
   - Only keep simhei.ttf (9.7MB) in project root
   - APK size reduced by ~9.7MB
